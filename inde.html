Plan de Trabajo Semanal – Desarrollo de Tesis

Este documento define la distribución de actividades para la redacción del proyecto de tesis, así como los lineamientos básicos para evitar retrabajos y mantener coherencia académica, especialmente en el uso de referencias y citas.

Capítulo 1 – Planteamiento del Problema

Responsable: Jesús

  • Redactar y mejorar el planteamiento del problema.
  • Definir claramente:
    • Problemática
    • Justificación
    • Objetivos (general y específicos)
    • Hipótesis
    • Alcance del proyecto
  • Preparar la estructura del proyecto para una presentación de 20–30 minutos.
  • Proponer el orden de diapositivas y el mensaje clave de cada sección.

Capítulo 2 – Estado del Arte

Responsable: Antonio

  • Investigar trabajos previos relacionados con:
    • Sistemas de respaldo energético
    • Inversores SPWM y SVPWM
    • Soluciones comerciales y académicas existentes
  • Identificar:
    • Qué se ha hecho
    • Quién lo ha hecho
    • Qué enfoques técnicos se han utilizado
    • Limitaciones de los trabajos existentes
    • Brecha tecnológica que aborda esta tesis
  • Usar el prompt de investigación proporcionado y fuentes académicas confiables.

Capítulo 3 – Marco Teórico

Responsable: Kevin

  • Antes de redactar:
    • Definir claramente qué conceptos son indispensables para el proyecto.
    • Evitar contenido genérico o de relleno.
  • Delimitar el marco teórico a:
    • SPWM
    • SVPWM
    • Conceptos directamente relacionados con el diseño y análisis del inversor
  • No iniciar la redacción formal hasta:
    • Tener claro el esquema del marco teórico.
    • Comprender el sistema de citas y referencias.

Lineamientos Generales para Referencias y Citas

  • Todo el equipo debe usar Zotero para gestionar referencias.
  • No se deben escribir citas manuales en Word.
  • Las citas deben insertarse siempre usando el complemento de Zotero.
  • El formato base será APA, pero debe mantenerse la compatibilidad para cambiar a IEEE si se solicita.
  • Las referencias incompletas pueden corregirse posteriormente en Zotero sin afectar el documento.
